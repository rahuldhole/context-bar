# Changelog

All notable changes to the "Context Bar" extension will be documented in this file.

## [0.2.0] - 2026-07-13

### Added
- Contribution guidelines and distribution bundle generation.
- SEO metadata and front matter to README.

### Changed
- Migrated to modern ESLint flat config and improved type safety in webview content.
- Simplified webview column resolution to default to ViewColumn.Active.
- Moved metadata from README to site configuration in `_config.yml`.

## [0.1.0] - 2026-04-19

### Added
- Initial release of Context Bar.
- Dynamic editor title bar buttons based on file extensions.
- Minimalist Swiss-design configuration GUI.
- Searchable command and keybinding mapper.
- Support for over 30 native VS Code icons.
- Instant pre-computed context matching for high performance.
